关于树的总结

发表于 2019-03-03

二叉树

概况

  1. 完全二叉树：二叉树中节点的编号和满二叉树中重合
  2. 满二叉树：所有的叶子节点满足都在最后一层，各层的结点数达到最大值
  3. 二叉树遍历主要有三种：
    • 先序遍历（根-左-右）
    • 中序遍历（左-根-右）
    • 后序遍历（左-右-根）
  4. 深度优先搜索算法（DFS）和广度优先搜索算法（BFS）
    • 广度优先搜索算法也是树的层次遍历，通俗地讲：就是一层层地遍历树的结点（实现的数据类型是queue，在java中是LinkedList类实现
    • 深度优先搜索算法，也可以看成是二叉树的中序遍历（实现的数据类型是stack，在java中是Stack类）

实现代码

  • BFS（广度优先搜索算法）

    • 思想：实现对二叉树一层一层地搜索遍历，所以，先将根节点进入队列（队首），然后进入循环，循环中先将队列中队首元素出队并且打印，然后判断他的左右子节点是否为空，不为空就分别入队，循环结束条件是直到队列为空。这个过程相当于是不断把节点一层层地放进队列中，因此可以按照层次顺序出队，实际情况用纸笔演练一下就是很简单的过程了。

      public void levelOrder(TreeNode node){
    LinkedList<TreeNode> queue = new LinkedList<TreeNode>();
    if(node == null){
        return;
    }else{
        queue.add(node);
        while(!queue.isEmpty()){
            //队列不为空
            TreeNode topNode = queue.poll();
            System.out.print(topNode.val + " ");
            if(topNode.left != null){
                queue.add(topNode.left);
            }
            if(topNode.right != null){
                queue.add(topNode.right);
            }
        }
    }

}

  • DFS（深度优先搜索算法）

    • 思想：其实思想和BFS很像，但是利用的数据结构是栈（Stack），先将根节点进栈，然后进入循环，循环首先将栈中最顶pop出，然后查看右节点和左节点是否存在，如果存在，则先后进栈，利用栈先进后出的特点，可以实现先遍历根节点，再遍历左节点，最后遍历右节点。用递归实现就是中序遍历。
public void DepthSearch(TreeNode node){
    Stack<TreeNode> stack = new Stack<TreeNode>();
    if(node == null){
        return;
    }else{
        stack.push(node);
        while(!stack.isEmpty()){
            TreeNode topNode = stack.pop();
            System.out.print(topNode.val + " ");
            if(topNode.right != null){
                stack.push(topNode.right);
            }
            if(topNode.left != null){
                stack.push(topNode.left);
            }
        }
    }

}

  • Z型的层次遍历（基于广度搜索）

    • 思想：

      • 其实整体思路是基于广度搜索的，但是用的是深度优先搜索的数据结构栈。
      • 这里需要用到三个栈，其中两个栈用来做替换，实际上：在遍历过程中，一个栈中的元素总是满足在同一层上。第三个栈是用来在前面两个栈进行角色切换时的temp栈。最后我们还需要一个指标isStartFromLeft，表示是否先获取左节点，再获取右节点。
      • 一开始还是先将根节点入栈（nowStack），然后进入循环，先将nowStack中的顶部元素出栈，根据isStartFromLeft决定先左后右还是先右后左。当nowStack空了，表明该层元素遍历完毕，这时候需要进行栈的切换，将nextStack变成nowStack，同时指针做相反改变（如true变成false）。

      • 循环结束的条件是：当nowStack和nextStack都为空时。

        public List<List<Integer>> zigzagLevelOrder(TreeNode root) {
    //同一个栈里面的都是同一层的元素
    Stack<TreeNode> temp = new Stack<TreeNode>();
    Stack<TreeNode> stackNow = new Stack<TreeNode>();
    Stack<TreeNode> stackNext = new Stack<TreeNode>();
    if(root == null){
        return new ArrayList<>();
    }else{
        stackNow.push(root);
        boolean isStartFromLeft = true;
        List<List<Integer>> resList = new ArrayList<List<Integer>>();
        List<Integer> list = new ArrayList<Integer>();
        while(!stackNow.isEmpty() || !stackNext.isEmpty()){
            TreeNode node = stackNow.pop();
            list.add(node.val);
            //System.out.print(node.val + " ");
            if(isStartFromLeft){
                //先将左节点进栈
                if(node.left != null){
                    stackNext.push(node.left);
                }
                if(node.right != null){
                    stackNext.push(node.right);
                }
            }else{
                //先从右节点进栈
                if(node.right != null){
                    stackNext.push(node.right);
                }
                if(node.left != null){
                    stackNext.push(node.left);
                }    
            }
            if(stackNow.isEmpty()){
                //每次清空表示换层
                isStartFromLeft = !isStartFromLeft;
                temp = stackNow;
                stackNow = stackNext;
                stackNext = temp;
                resList.add(list);
                list = new ArrayList<Integer>();
            }
        }
        return resList;
    }
}
